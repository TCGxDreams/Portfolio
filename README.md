# Nguyễn Vũ Trọng Nhân // F1 Telemetry Portfolio

A premium, high-performance web portfolio inspired by F1 telemetry systems and dashboards. Developed for **Nguyễn Vũ Trọng Nhân**, showcasing expertise in Bioinformatics, Graph Neural Networks (GNNs), and AI for Drug Discovery.

## 🏎️ Core Features

- **F1 Telemetry HUD Styling**: A cutting-edge, dark-themed UI featuring scanlines, carbon meshes, and glowing cyan/lime accents simulating a high-end racing telemetry dashboard.
- **Interactive GNN Simulator**: An animated, spring-physics-based Graph Neural Network simulator demonstrating protein-ligand binding pocket modeling, adjustable layer depth, and active node feedback.
- **Dynamic System Diagnostics**: Real-time mock telemetry streams, CPU load matrices, and interactive G-Force radar components.
- **Professional Stint Tracking**: Visual timelines mapping out professional experience, achievements, awards, and educational milestones using racing metrics (`LOAD`, `TEMP`, `LAP`).
- **Responsive Architecture**: Built on modern web standards ensuring fluid compatibility across desktop and mobile terminals.

## 🛠️ Technology Stack

- **Framework**: React.js 19
- **Build Tool**: Vite
- **Styling**: Custom vanilla CSS (`main.css`, `telemetry.css`) featuring modern CSS variables, Grid, and Flexbox layouts.
- **Animations**: CSS Transitions, Keyframes, Web Animations API, SVG Manipulation.
- **Audio**: Web Audio API for synthetic digital triggers and confirm blips.

## 🚀 Quick Start (Local Terminal)

1. Clone the repository:
   ```bash
   git clone https://github.com/TCGxDreams/Portfolio.git
   cd Portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Boot the local telemetry server:
   ```bash
   npm run dev
   ```
4. Build for production deployment:
   ```bash
   npm run build
   ```
